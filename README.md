# Tina Jain - Job Search Agent

This repository contains the AI agent and dashboard for scanning job opportunities, evaluating convertibility, and identifying potential references (alumni, connections, bosses) for **Tina Jain**.

## Setup Instructions

1. **Paste CV PDF**: Place the CV PDF file in this directory.
2. **Configure API Keys**: Set up your Google Gemini API key or search API keys as required in `.env`.
3. **Run Agent**: Run the script to perform the search and analysis.
4. **View Dashboard**: Open the generated HTML page to search, filter, and review the curated opportunities and connection lists.
